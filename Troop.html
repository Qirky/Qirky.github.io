    About Me

    Troop is a piece of software that not only allows multiple users to edit the same body of code simultaneously, but also lets users interact with their program and "run" parts of their program while its being edited. This last part is part of the nature of live coding and Troop makes it easier for users to do it together.

    For installing and running Troop, see the GitHub page page, which goes in to some more detail on the topic.
