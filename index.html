    About Me

    My name is Ryan Kirkbride and I like making music with code: sometimes melodic, sometimes ambient, sometimes techno - always algorithmic. I use a custom written Python library for Live Coding called FoxDot, which is a bit like a mini language inside of Python for making music; check out the rest of my GitHub for more info!
